# Kho lưu trữ Bài tập Môn học DSA++ (Cấu trúc Dữ liệu và Giải thuật Nâng cao)

Đây là kho lưu trữ cá nhân (repository) chứa mã nguồn và các báo cáo bài tập cho môn học DSA++ (Cấu trúc dữ liệu và Giải thuật Nâng cao).

## Nội dung thư mục

### 1. `CS523_Decuong.pdf`
Đề cương chi tiết của môn học DSA++.

### 2. Thư mục `homeworks/`
Chứa các bài tập về nhà trong quá trình học tập.

#### 2.1. Nhóm bài tập: `Ứng Dụng CTDL`
Nội dung: Trình bày, giải thích và phân tích 3 Cấu trúc dữ liệu và 3 Giải thuật phổ biến trong:
- Máy tính / Hệ điều hành (OS)
- Hệ quản trị CSDL (DBMS)
- Hệ thống thông tin địa lý (GIS)
*Sản phẩm đính kèm: Bài luận LaTeX nội bộ (`Baitap_UngDungCTDL.tex`).*

#### 2.2. Nhóm bài tập: `Tạo ứng dụng minh họa sắp xếp ngoại`
Nội dung: Ứng dụng mô phỏng trực quan thao tác của giải thuật Sắp xếp Ngoại (External Sort) thông qua kỹ thuật K-Way Merge Sort trên đĩa từ với dữ liệu số thực.
*Sản phẩm đính kèm:*
-   `app.py`: Mã nguồn ứng dụng giao diện (Tkinter GUI).
-   `external_sort.py`: Lõi thuật toán xử lý phân vùng file nhị phân.
-   `HuongDanSuDung.tex`: Hướng dẫn vận hành ứng dụng.
-   `plans/`: Thư mục lưu vết 10 bước lên kế hoạch cấu trúc hệ thống.

## Hướng dẫn 
Để kiểm tra bài tập Ứng dụng Sắp xếp ngoại:
1. Đảm bảo Python 3 được cài đặt.
2. Mở terminal tại thư mục `homeworks/Tạo ứng dụng minh họa sắp xếp ngoại`.
3. Chạy lệnh: `python app.py`
